Landuse Construction

Description

This map adds a marker icon to any 'ways' tagged as landuse=construction. Since the results are ways we use the centre point of the way for the marker.

Frequency

This map is re-generated every 3 days.

Overpass Query

area["name"="Ireland"]->.boundaryarea; ( way(area.boundaryarea)[landuse=construction]; ); out center;
